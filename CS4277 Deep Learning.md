# CS4277: Deep Learning

Instructor: Chen Zhao, czhao4@kennesaw.edu

## 📘 Course Overview
CS4277 provides a comprehensive introduction to **deep learning**, covering foundational concepts, modern neural network architectures, and advanced topics such as generative models and large-scale training. The course combines theoretical understanding with hands-on implementation using tools like NumPy, Pandas, PyTorch, and TensorFlow.
